moveAllToSameDocument

public static int moveAllToSameDocument(ExtentIterator iterators[0..*])
Moves all iterators in the array to the same document. This method will only stop at documents where all of the iterators have a match. This code assumes that the most selective iterator (the one with the fewest document matches) is the first one. @return The document number that the iterators are now pointing to, or Integer.MAX_VALUE if one of the iterators is now done.
