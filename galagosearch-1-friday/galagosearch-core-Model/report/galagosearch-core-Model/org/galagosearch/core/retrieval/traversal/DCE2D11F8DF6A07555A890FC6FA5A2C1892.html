public class ImplicitFeatureCastTraversal
implements Traversal

For many kinds of queries, it may be preferable to not have to type an explicit #feature operator around a count or extents term. For example, we want #combine(#feature:dirichlet(#counts:dog())) to be the same as #combine(dog). This transformation automatically adds the #feature:dirichlet operator. @author trevor
