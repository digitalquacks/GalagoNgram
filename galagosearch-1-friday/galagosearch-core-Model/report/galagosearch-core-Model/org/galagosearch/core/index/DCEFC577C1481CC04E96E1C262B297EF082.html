public class BackedCompressedByteBuffer

A BackedCompressedByteBuffer is like a CompressedByteBuffer, but it can overflow into disk storage if necessary. Unlike a CompressedByteBuffer, there's no getBytes() method since that wouldn't makes sense if all the data is on disk. @author trevor
