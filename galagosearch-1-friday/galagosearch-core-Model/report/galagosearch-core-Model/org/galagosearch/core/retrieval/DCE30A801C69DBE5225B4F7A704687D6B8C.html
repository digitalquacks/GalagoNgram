documentToScore

public static int documentToScore(int document, int maxScore, int maxBits, int mask)
The goal here is to take in a number that ranges from 1 to 2^maxBits, then turn that into a score. The score needs to be nicely distributed so that there are a few maximum values and lots of minimum values. To do this, we flip the document number backwards so that the high bit is now the low bit. This is kind of like a random number generator, but it's 1-to-1 with the input number. Then, we take the log of that number with respect to the maximum. Most of these log numbers will be high, and a few will be low, so we map low log numbers to high score values.
